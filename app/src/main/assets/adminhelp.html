Settings - Personalising the App

Settings

Use this to set up the items of Income and Expenditure in your Plan.

Move between categories by selecting from the menu options to the left.

Select all your sources, for example 'Wages' by ticking the boxes next to them.

If you wish to add new items of Income or Expenditure, type the name you desire in the box to the left (headed 'Add Another Source') and select 'Add'. These will then be added to the bottom of the existing list. Again, select the new sources you create by ticking the boxes next to them.

When you have completed setting up, select the 'Settings' icon again to return to input mode.
