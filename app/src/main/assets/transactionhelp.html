Editing Data

Use this operating mode to make any desired changes to your original input.

Edit Item

Select any boxes marked with the 'Pencil' icon and make your desired changes.

Select the 'trash can' icon at the end of those input lines you would like to delete completely. When prompted, choose 'Yes' to confirm.

Save

Press 'Save' against each line you have changed or deleted. NB You must press 'Save' at the end of each line you have changed or deleted.

If you wish to add a completely new line, first save the changes you have already made by pressing 'Save' and then press '+' to return to input mode. Finally, press '+' again to add the new input line.
