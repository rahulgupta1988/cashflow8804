Inputting Data

Add Item

Press '+' to add a new input line From the first drop-down box, select an item, for example 'Wages'.

From the second drop-down box, select the transaction frequency, for example 'Every Week'. In the next box, enter the amount (a monetary value). In the final drop-down box, choose from the calendar the date the receipt or payment will next occur.

If you would like to add a comment, select the 'Pencil' icon after the date box, type in your comment and press 'Add Comment'.
Save Press 'Save' to enter your transaction. NB You must press 'Save' at the end of each input line.
